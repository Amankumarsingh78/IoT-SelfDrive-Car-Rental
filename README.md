# 🚗 IoT Self-Drive Car Rental Platform

A production-grade Full Stack IoT-enabled Self-Drive Car Rental Platform built using React, Node.js, Express, and MySQL.

## 🚀 Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MySQL

### Authentication
- JWT
- bcrypt

### Features
- User Authentication
- Car Booking
- KYC Verification
- Admin Dashboard
- IoT Vehicle Simulation
- GPS Tracking
- Dynamic Fare Calculation
- Revenue Analytics
